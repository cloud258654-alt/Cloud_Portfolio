# Antigravity Stock AI Ver 2.1 更新紀錄

更新日期：2026-06-24

## 更新目標

Ver 2.1 參考 TradingView、Finviz、Koyfin、Seeking Alpha、Morningstar、StockAnalysis 等國外投資研究平台，將 Ver 2.0 的市場總覽延伸為更完整的研究工具台。

本版重點是補上國外成熟投資工具常見的五個能力：

- 熱力圖
- 進階篩選器
- 個股比較
- 投資組合健康檢查
- 警示與回測入口

## 已完成更新

1. 新增「國際級研究工具台」
   - 放在首頁市場總覽下方。
   - 作為 Ver 2.1 的主要功能入口。
   - 保持靜態部署，不需要後端即可展示完整產品方向。

2. 新增「台股產業熱力圖」
   - 參考 Finviz Maps 的市場掃描體驗。
   - 以顏色顯示產業漲跌強弱。
   - 以區塊大小呈現產業權重與成交比重概念。
   - 點擊產業區塊可直接啟動代表個股分析。

3. 新增「進階篩選器」
   - 參考 Koyfin / Finviz / Seeking Alpha 的股票篩選器架構。
   - 顯示 AI 分數、排名與條件標籤。
   - 篩選條件包含 EPS YoY、高 ROE、外資回補、成交量放大、AI 供應鏈、低估值、法人連買等。

4. 新增「個股比較」
   - 參考 TradingView Fundamental Graphs 與 Seeking Alpha Comparisons。
   - 支援多檔股票並排比較。
   - 比較欄位包含 P/E、ROE、營收 YoY、籌碼狀態、AI 評分。

5. 新增「投資組合健康檢查」
   - 參考 Seeking Alpha Portfolio Health Check。
   - 顯示 Portfolio Health 分數。
   - 檢查項目包含產業集中度、波動風險、股利穩定度、AI 平均評分。

6. 新增「警示與回測入口」
   - 參考 TradingView Alerts 與 Strategy Testing。
   - 目前提供功能入口展示：
     - 價格警示
     - 訊號警示
     - 策略回測
   - 後續可接後端或瀏覽器通知做真正提醒。

7. 首頁流程優化
   - 進入個股分析或全球總經分析時，Ver 2.1 工具台會自動隱藏。
   - 返回首頁時重新顯示市場總覽與工具台。

8. 手機版樣式補強
   - 熱力圖在手機改為雙欄。
   - 個股比較表支援水平捲動。
   - 工具台模組在手機改為單欄排列。

## 目前仍為靜態模擬資料

以下項目目前仍為本地模擬資料：

- 產業熱力圖。
- 進階篩選器結果。
- 個股比較數據。
- 投資組合健康分數。
- 警示與回測條件。

設計上已保留資料模型，後續可用 API 或每日資料檔替換。

## 建議 Ver 2.2 優先項目

1. 將進階篩選器改為可操作條件面板
   - 支援勾選條件。
   - 支援多條件交集。
   - 支援排序與重設。

2. 將熱力圖接入真實產業資料
   - 面積：成交值或市值。
   - 顏色：漲跌幅。
   - Tooltip：產業成交比重、主力個股、資金流向。

3. 建立真正的個股比較頁
   - 支援使用者自行加入 2 到 4 檔股票。
   - 加入財務趨勢圖、估值百分位、股利、籌碼與風險。

4. 建立本地警示規則
   - 使用 `localStorage` 儲存規則。
   - 支援價格、AI 評分、法人連買、均線突破等條件。

5. 建立簡易回測模擬器
   - 使用歷史模擬 K 線。
   - 顯示勝率、平均報酬、最大回撤、停損停利效果。
