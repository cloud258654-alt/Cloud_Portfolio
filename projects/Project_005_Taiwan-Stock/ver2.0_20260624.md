# Antigravity Stock AI Ver 2.0 更新紀錄

更新日期：2026-06-24

## 更新目標

本次 Ver 2.0 以「從展示型 Demo 走向每日可用的台股研究工具」為目標，參考 Goodinfo、WantGoo、TradingView、TWSE 等同類型網站常見資訊架構，補強市場總覽、選股入口、個人化清單、資料來源與 AI 決策可信度。

## 已完成更新

1. 首頁新增「台股市場總覽」
   - 加權指數、櫃買指數、電子類股、金融保險指標卡。
   - 顯示漲跌點、漲跌幅、成交金額或市場狀態。
   - 目前使用靜態模擬資料，保留未來 API 接入位置。

2. 新增「產業強弱」模組
   - 顯示產業漲跌與強弱條。
   - 讓使用者不用先輸入個股，也能快速掌握市場資金方向。

3. 新增「智能選股條件」
   - 提供法人連買、營收成長、高殖利率、成交量異常、AI 供應鏈等快速入口。
   - 點擊條件會直接帶入代表性個股並啟動分析流程。

4. 新增「最近搜尋 / 自選股」
   - 使用 `localStorage` 儲存最近搜尋。
   - 可從最近搜尋加入自選股。
   - 不需要後端即可保留使用者的輕量操作紀錄。

5. 報告頁新增「AI 決策可信度與資料來源」
   - 顯示 AI 信心分數。
   - 顯示技術面、基本面、籌碼面、總經面、新聞事件的權重拆解。
   - 顯示資料來源規劃：TWSE、Goodinfo、MOPS、TradingView 篩選維度。
   - 明確標示目前為靜態模擬資料，避免誤解為即時行情。

6. 更新分析流程狀態
   - 進入個股分析或全球總經分析時，首頁市場總覽與推薦清單會隱藏。
   - 返回首頁時重新顯示市場總覽與推薦清單。

7. 響應式樣式補強
   - 市場總覽、產業強弱、選股條件、可信度卡片皆支援手機版單欄排列。

## 仍使用模擬資料的項目

- 市場指數與成交金額。
- 產業強弱。
- 智能選股條件。
- AI 信心分數。
- 資料來源列表。

這些項目目前主要用於產品架構展示。正式版本建議改接 API 或每日資料檔。

## 建議 Ver 2.1 優先項目

1. 接入 TWSE 官方資料
   - 三大法人買賣超。
   - 融資融券。
   - 借券資訊。
   - 市場成交統計。

2. 接入 MOPS / 公開資訊觀測站
   - 重大訊息。
   - 財報公告。
   - 法說會與股東會資訊。

3. 建立個股比較功能
   - 支援 2 到 4 檔股票並排比較。
   - 比較估值、成長、獲利率、籌碼、技術面與 AI 評分。

4. 強化篩選器
   - 支援多條件組合。
   - 加入殖利率、本益比、ROE、EPS 年增、營收 YoY、法人連買、成交量異常等條件。

5. 提升報告可驗證性
   - 每個 AI 結論旁加入對應數據依據。
   - 每個結論標示資料日期與來源。
   - 加入資料延遲與非投資建議提示。
