# 停車場收費管理流程系統開發規格書

## 1. 專案目標

請開發一套「停車場收費管理流程系統」MVP 版本。

第一版不需要真實串接硬體，先用 Web 系統模擬完整流程，未來可擴充車牌辨識攝影機、柵欄機、自助繳費機、電子發票與線上支付。

## 2. 使用技術

請使用以下技術：

- 前端：React + Vite
- UI：Tailwind CSS
- 後端：Node.js + Express
- 資料庫：SQLite
- ORM：Prisma
- 語言：JavaScript
- 執行環境：本機開發

## 3. 核心功能

### A. 管理員登入

- 管理員可以登入系統
- 預設帳號：admin
- 預設密碼：admin123
- 登入後才能進入後台

### B. 車輛進場管理

功能：

- 輸入車牌號碼
- 選擇車輛類型：臨停 / 月租 / VIP / 黑名單
- 記錄進場時間
- 自動建立停車紀錄
- 顯示目前場內車輛

進場流程：

```text
輸入車牌
→ 檢查是否黑名單
→ 檢查是否已在場內
→ 建立進場紀錄
→ 車位數 -1
→ 顯示進場成功
