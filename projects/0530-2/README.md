# 🪐 Cloud T - Premium 我們這一家午餐大轉盤與在地美食老虎機

歡迎來到我的個人品牌午餐抉擇主控台！這是一個專為現代辦公團隊設計、採用極致暗色玻璃擬態（Glassmorphism）與螢光極光（Aurora）風格打造的互動式網頁產品。

🌐 本地即時預覽 / Local Demo : [http://localhost:8000/index.html](http://localhost:8000/index.html)

## 🎨 介面視覺亮點 (Visual & Interactive Features)

1. **我們這一家經典卡通人物 SVG**：內建花媽、花爸、橘子、柚子、水島太太 5 大高精細向量圖，支援名單編輯時點擊「頭像選擇器」即時輪播切換，完全免除外部圖源失效之問題。
2. **擬真物理碰撞大轉盤 (Lucky Spin Wheel)**：結合三次方減速摩擦阻尼物理學與指針「彈簧-阻尼彈力回彈（Spring Damping）」物理效果，大轉盤慢速停下之過程極具視覺張力與抉擇懸念。
3. **微動態參賽看板 (Contestant Board)**：大轉盤下方設有精美磨砂玻璃質感的選手徽章，轉盤飛速旋轉時，參賽徽章會與轉盤實體指針碰撞動作同步點亮發光，提供卓越的視覺反饋。
4. **Web Audio 物理音效合成 (Audio Synthesis)**：零外部音訊檔案依賴，由瀏覽器底層即時演算合成指針碰分隔線時的清脆「滴答聲」，與中獎得主定格時上揚的「獲勝 C 大調琶音」。
5. **極客風美食老虎機 (Neon Food Slot Machine)**：將抽中之跑腿買單人一鍵導向「吃什麼」環節，支援 1-5 品項防呆。美食卡片在老虎機中飛速滾動，並以三次方曲線自然減速定格，伴隨清脆木琴聲。
6. **雙重卡漫大結局 (Combined Victory Presentation)**：彈出豪華毛玻璃擬態對話框，左側展示「大跳勝利搖擺舞之得獎頭像」，右側展示「動態蒸汽上升與蛋黃抖動美食插圖」，並在空中發射繽紛五彩碎紙花（Canvas Confetti）。
7. **全方位響應式網頁設計 (RWD)**：極佳的螢幕自適應排版，在手機、平板與寬螢幕桌機均有完美的卡片縮放與版面配置。

## 📤 如何部署至 GitHub Pages 線上分享

如果您要在 GitHub 上建立線上 Demo：

1. **建立 GitHub 倉庫**：在 GitHub 上新建一個名為 `0530-2` 的公開倉庫（Public Repository）。
2. **上傳檔案**：將 `index.html` 與 `README.md` 上傳至該倉庫。
3. **啟用 Pages**：
   - 進入該倉庫的 **Settings** ➡️ **Pages**。
   - 在 **Build and deployment** 下方的 Branch 選擇 `main` (或 `master`)，資料夾選擇 `/ (root)`，點擊 **Save**。
4. **線上遊玩**：約等候 1-2 分鐘後，即可透過 `http://localhost:8000/index.html` 本地暢玩並分享給同事朋友！
