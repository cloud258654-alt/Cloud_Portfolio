# UI UX Guideline

## Visual Direction

The interface should feel quiet, premium, and operational: Apple-level clarity, Toyota-like reliability, Notion-style organization, Linear-style focus, and a film production tool sensibility for Google Flow workflows.

## Palette

- Background: `#F8F7F3`
- Surface: `#FFFFFF`
- Surface Muted: `#F3F1EC`
- Text: `#111827`
- Muted Text: `#6B7280`
- Accent Red: `#E11D2E`
- Soft Blue: `#DCEBFF`
- Warm Wood: `#E8D8C3`
- Border: `#E5E7EB`
