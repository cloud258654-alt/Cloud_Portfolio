# Google Flow Enterprise Director OS PRD

## Purpose

Google Flow Enterprise Director OS is a local-first PWA for planning AI filmmaking work destined for Google Flow. It turns project inputs into structured storyboards, production bibles, scene prompts, voice over, subtitles, QA checks, and exports.

## Core Modules

- Dashboard
- Projects
- Storyboard
- Project Bible
- Characters
- Vehicles
- Environments
- Scenes
- Prompts
- Voice
- Subtitles
- QA Center
- Export
- Settings
