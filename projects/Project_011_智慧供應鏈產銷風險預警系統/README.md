## 🌸 設計理念 / Design Philosophy (Calm Technology)
本系統採用 **日系簡約高端企業級設計語彙 (Japanese Premium Minimalist UI/UX)**，完美融合 **「Apple 官網風格 + MUJI 簡約感 + 日式溫暖咖啡廳」** 的視覺美學：

*   **靜態科技感 (Calm Technology)：** 屏棄傳統工業或 ERP 系統冰冷擁擠的黑色與深藍，改用高留白、呼吸感極佳的溫暖卡其灰 (`#F7F8FA`) 與純白背景。
*   **障子木格柵紋理 (Shoji Grid Background)：** 於背景融入低透明度（1% 濃度、120px 間距）的溫潤木質十字細線，營造出傳統日式障子木門或塌塌米的沉穩安定氛圍。
*   **木質自然色系 (Natural Timber Accent)：** 以炭黑 (`#1A1A1A`) 作為高對比標題字體，搭配溫和橡木棕 (`#7B5E45`) 作為品牌核心亮點，並將狀態色（紅、黃、綠）全面軟化為大地色系的自然原色。
*   **宣紙感實時控制台 (Warm Paper Terminals)：** 將傳統工程師黑底綠字的命令行控制台，全面重新設計為復古宣紙溫潤感 (`#F5EFE6` 底色、天然橡木邊框與深炭灰 Monospace 字體)，實現溫潤典雅的印刷日誌質感。
*   **人文插飾擺設 (Calm Tech Decors)：** 於介面邊緣融入純 CSS 繪製的懸浮木質置物架、黏土陶器花瓶、乾燥樹枝與蒲公英插花，為冷冰冰的數據中心導入咖啡廳的人文溫度。

## 🚀 核心功能模組 / Core Feature Sets

### 📊 1. BI 決策控制塔 (BI Operational Dashboard)
*   **Ver 2.0 營運價值調優：** 展示控制塔介入後的卓越改善成效：庫存天數 (DOH) 由 120 天降低至 **75 天**、呆滯料金額降低至 **$12,000 USD**，預估節省採購資金達 **$18,400 USD**。
*   **庫存健康矩陣：** 以環形數據圖表即時呈現 IC 晶片、PCB、被動元件等各分類的動態防護係數。
*   **週變動智慧週報：** 支援一鍵呼叫 AI 主動式營運週報，穿透庫齡呆滯結構與根因分析。

### 🗺️ 2. 全球風險雷達與事件模擬 (Global Risk Radar & Event Engine)
*   **全球多維風險監控：** 即時追蹤美國加徵 12% 晶片關稅、美西港口塞港、電解銅價格暴漲與新竹地震等全球突發事件。
*   **事件驅動 AI 協同工作流：** 內建模擬引擎，觸發突發關稅事件後，實時演示 **「事件捕捉 ➔ 缺料代理 ➔ 合約代理 ➔ 替代料代理 ➔ 採購代理 ➔ 決策完成」** 的多 Agent 自動串聯瀑布流。

### 🕸️ 3. 多層級供應網拓撲 (Multi-tier Supplier Network Graph)
*   **三層級依賴穿透：** 以互動式響應型 SVG 拓撲圖繪製 T1（意法半導體、樂鑫）、T2（台積電晶圓廠、日月光封測廠）至 T3（德國晶圓矽片）的階層式供應依賴鏈。
*   **斷鏈風險預警：** 點擊節點即可查閱各供應商的 ESG 評分、交期波動，以及針對封測廠火警等事件的應變替代計畫。

### 🔥 4. AI 風險戰情室與知識圖譜 (AI Risk War Room & Knowledge Graph)
*   **重大事件損害評估：** 針對中國工廠停工、美金暴漲等重度危機進行營運曝險金額（高達 $450,000 USD）、波及產品線與受害合約客戶的精準量化計算。
*   **多維度企業知識圖譜：** 連接並渲染 `供應商 ➔ 零件 ➔ 倉庫 ➔ 採購單 ➔ 產品 ➔ 客戶 ➔ 營收` 的端到端水平傳導路徑。
*   **可解釋性 AI 決策面板 (XAI)：** 拒絕黑盒決策，完整列出 AI 應變方案背後的所有客觀物理證據（價格對比、空運交期、歷史替代成功率）。
*   **人機協同閉環審核流 (Human-in-the-Loop)：** 滿足上市公司內控合規審查。主管點擊「核准決策並自主執行」後，觸發模擬 ERP 工作流，自動建立 PO、寄發緊急催料郵件，並鎖定進入實時物流軌跡追蹤。

### 🤖 5. AI 員工管理中心 (AI Agent Hub)
*   **7 位自主 AI 員工：** 集中監控 Inventory、Shortage、Supplier、Contract、Alternate、Procurement、Risk 等 7 位 AI 代理的運作效能。
*   **營運效能計量表：** 實時追蹤各 Agent 的決策準確率指標（平均達 97.9%）、本月處理任務次數，以及精確計量 Token 消耗與月度營運成本。
*   **實時日誌終端：** 展開卡片即可穿透查閱各 Agent 底層邏輯推理日誌（Monospace Console Logs）。

### 📂 6. 合約 RAG 知識庫與資料倉儲 (RAG Vector Database & PostgreSQL Monitor)
*   **RAG 語義塊檢索：** 展示合約上傳與 PDF 向量化分塊（Chunking）結果，提供對合約違約賠償條款（如每日延誤扣款 1.5%）的智慧語義定位。
*   **PostgreSQL 監控端：** 內置數據倉庫瀏覽器，允許開發者查閱 `war_room_scenarios`、`agent_hub_metrics` 與 `global_risk_radar` 等核心表格的實體 Schema 欄位與模擬列資料。

## 💻 本地端運行指南 / How to Run Locally

### 1. 前置準備 / Prerequisites
請確保您的開發環境已安裝 **Node.js (建議 v18.0 或以上版本)** 以及 **npm** 套件管理器。

### 2. 複製專案與安裝 / Clone & Install
```bash
# 複製儲存庫 (如果您在多專案儲存庫中，請切換至本專案目錄)
cd projects/Project_011_智慧供應鏈產銷風險預警系統

# 安裝相關依賴套件
npm install
```

### 3. 啟動開發伺服器 / Start Development Server
```bash
# 啟動本地 Vite 開發伺服器
npm run dev
```
啟動後，請於瀏覽器輸入 **`http://localhost:5173/`** 即可進入系統進行本地操作。

### 4. 生產環境打包與預覽 / Build & Preview
```bash
# 編譯打包靜態資源 (生成 dist 目錄)
npm run build

# 本地預覽生產環境打包網頁
npm run preview
```
啟動預覽後，請於瀏覽器輸入預設網址 **`http://localhost:4173/`** 進行瀏覽。
