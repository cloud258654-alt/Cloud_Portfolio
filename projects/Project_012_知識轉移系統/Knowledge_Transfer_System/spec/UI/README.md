# UI

User interface specifications and design contracts.

## Files

- `Design_System.md`: Enterprise AI design language and component rules.
- `design-tokens.json`: Framework-neutral design tokens.
- `design-tokens.css`: CSS variable implementation.
- `tailwind.preset.js`: Tailwind preset for future frontend implementation.
- `UI_M01_Document_Knowledge_Center.md`: Document module UI.
- `UI_M02_AI_QA_Assistant.md`: AI QA module UI.
- `UI_M03_Experience_Transfer.md`: Experience transfer module UI.

Frontend implementations should consume the tokens before adding bespoke styling.
