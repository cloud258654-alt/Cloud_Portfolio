# Sprint 01: Authentication

## Objective

Implement the authentication foundation for users, sessions, roles, and secure API access.

## Scope

- User account model
- Login and logout flow
- JWT access token handling
- Role-based access control baseline
- Authentication API tests

## Deliverables

- Users can authenticate
- Protected APIs reject unauthenticated requests
- Roles can be assigned and checked
- Authentication behavior is documented
