# 5 分鐘展示腳本 — 福田貨櫃倉儲管理系統 v0.5.1

> 本腳本設計給展會、客戶簡報或內部 Demo 使用，全程 5 分鐘。

---

## 第 1 分鐘：Dashboard 營運總覽

**展示頁面**：Dashboard

**操作步驟**：
1. 開啟系統 → 預設進入 Dashboard
2. 指向最上方 12 張 KPI 數字卡

**要講的重點**：
> 「這是貨櫃倉儲的管理大盤。一眼可以看到本期應收多少、已收多少、還有多少沒收。逾期金額、空櫃數量、維修中貨櫃都一目瞭然。」

**預期觀眾看到**：
- 12 張卡片以綠/紅/黃/藍色標示不同狀態
- 系統日期為當天實際日期

---

## 第 2 分鐘：客戶 → 預約 → 合約

**展示頁面**：客戶管理 → 預約管理 → 電子合約

**操作步驟**：
1. 點「客戶管理」→ 選一位客戶（如王先生）→ 展示客戶卡片（貨櫃/預約/合約/帳單）
2. 點「預約管理」→ 展示等待名單 + 保留貨櫃下拉選單 + 訂金狀態
3. 點「電子合約」→ 展示合約 Kanban + 表格 + 條款檢視

**要講的重點**：
> 「從客戶進來到簽約，系統全程追蹤。預約時可指定貨櫃，收訂金後合約頁會顯示預約訂金已收。合約條款包含租金、押金、電費、中途解約、退款規則，全部可查。」

**預期觀眾看到**：
- 客戶中心有完整的客戶 + 貨櫃 + 帳單關聯
- 預約表格有綠色「已收 $6,000」或紅色「待收訂金」
- 合約表格有 14 個欄位，右側可查看完整條款

---

## 第 3 分鐘：收費 → 電費 → 逾期

**展示頁面**：收費管理 → 電費管理 → Dashboard（回到風險提醒）

**操作步驟**：
1. 點「收費管理」→ 展示未收總額 + 年度帳單產生按鈕 + 部分付款紀錄（hover 看明細）
2. 點「電費管理」→ 展示電表度數 + 基本費/用電費拆分
3. 回到 Dashboard → 指向「營運風險提醒」區

**要講的重點**：
> 「帳單自動依合約類型計算年租金。部分付款可記錄多筆，系統自動加總。逾期帳單會出現在風險提醒，紅色標籤一看就知道要先處理誰。」

**預期觀眾看到**：
- 付款方式下拉 + 標記已繳按鈕
- P005 有 2 筆部分付款（hover 顯示明細）
- 收據/發票狀態 badge
- 風險提醒列出逾期客戶 + 金額

---

## 第 4 分鐘：退租試算 → 結案

**展示頁面**：退租管理

**操作步驟**：
1. 進入「退租管理」
2. 選擇一個貨櫃（如 A01 王先生）
3. 設定退租日期（如 2026-07-31）
4. 輸入損壞扣款 2000、清潔費 500、違約金 0
5. 展示試算結果：扣款合計 → 最終應退/應補
6. 展示「退租結案摘要」15 項明細
7. 點「完成退租」→ 展示 Kanban 變更為「已完成」

**要講的重點**：
> 「退租試算自動帶入合約資料，計算已用月份、剩餘租金退費。押金先扣掉未繳費、電費、損壞、清潔費、違約金，最後算出要退多少或補多少。結案後貨櫃自動釋出、合約關閉、客戶狀態自動更新。」

**預期觀眾看到**：
- 試算表 13 行明細（含扣款合計 + 最終應退）
- 結案摘要包含結案後三狀態（container: available, contract: closed, customer: closed）
- Kanban 卡片移至「已完成」欄

---

## 第 5 分鐘：AI 助理 + 匯出報表

**展示頁面**：AI 助理 → Dashboard 資料管理

**操作步驟**：
1. 點右下角浮動 AI 按鈕 → 快速查看摘要
2. 點「開啟完整助理」→ 輸入「今天要處理什麼？」→ 展示編號待辦清單
3. 輸入「營運風險有哪些？」→ 展示風險統計
4. 回到 Dashboard → 點「Export JSON」→ 下載備份檔
5. 點「CSV: Customers」→ 下載 Excel 可開的客戶清單

**要講的重點**：
> 「AI 助理可以回答空櫃、未繳款、逾期、收入、合約到期、待辦事項等營運問題。所有資料可以一鍵匯出 JSON 備份或 CSV 報表，Excel 可以直接打開。系統會自動儲存到瀏覽器，關掉重開資料不會消失。」

**預期觀眾看到**：
- AI 回答具體的客戶名稱、金額、建議動作
- JSON 下載成功
- CSV 用 Excel 開啟後中文正常顯示
