# 使用者操作指南 — 福田貨櫃倉儲管理系統 v0.5.1

> 本文件適用於非技術背景的系統操作人員。

---

## 一、Dashboard 營運總覽

打開系統後第一個看到的畫面就是大盤（Dashboard）。

### 怎麼看

1. **最上方數字卡**：12 張卡片顯示「本期應收租金」「已收金額」「未收金額」「逾期金額」「空櫃數」等重要數字。
   - 綠色 = 良好（如已收款、空櫃可用）
   - 紅色 = 需注意（如逾期、未收）
   - 黃色 = 提醒（如即將到期）

2. **營運風險提醒**：列出目前需要注意的項目，例如「某客戶逾期未繳」「某貨櫃合約 30 天內到期」。
   - 每筆都有紅色/黃色標籤，一看就知道優先順序

3. **今日待辦**：系統自動整理的待辦清單
   - 🔴 高優先：催收逾期帳單
   - 🟡 中優先：聯絡續約客戶
   - 🔵 低優先：一般追蹤
   - 點擊任何一項會跳到對應的功能頁面

4. **貨櫃狀態摘要**：一條橫條圖顯示目前有多少櫃子可租、已租、預約中、維修中

---

## 二、新增貨櫃

1. 左側選單點「貨櫃管理」
2. 右下方「新增貨櫃」表單輸入：
   - 貨櫃編號（例如：C02）
   - 規格（10 呎或 20 呎）
   - 區域（A / B / C）
   - 狀態（通常選「空櫃」）
3. 點「新增貨櫃」

---

## 三、新增客戶

1. 左側選單點「客戶管理」
2. 右側「新增客戶」表單填寫：
   - 必填：姓名、電話
   - 建議填：Line ID、地址、Email
   - 付款偏好（匯款/現金/支票/信用卡）
   - 通知偏好（LINE/電話/Email）
3. 點「新增客戶」

---

## 四、建立預約

1. 左側選單點「預約管理」
2. 查看目前的等待名單
3. 在等待中的預約旁，可以選擇一個空櫃再點「通知保留」
4. 若客戶已付訂金，表格中會顯示綠色「已收 $X,XXX」；未付則顯示紅色「待收訂金」

---

## 五、查看電子合約

1. 左側選單點「電子合約」
2. 上方是合約 Kanban（草稿 / 待簽署 / 已簽署）
3. 中間表格列出所有合約，包含：
   - 承租人姓名與電話
   - 貨櫃編號與規格
   - 合約類型（首年或續約）
   - 租金、押金、收款狀態
   - 若有預約訂金也會顯示（綠色金額）
4. 右側可選擇任一合約查看完整條款（承租人資料、貨櫃資料、租金押金、電費規則、中途解約、退款規則等）
5. 未簽署的合約可點「模擬簽署」按鈕

---

## 六、管理收費

1. 左側選單點「收費管理」
2. 上方數字卡顯示未收總額、逾期筆數
3. 點「產生年度租金帳單」會自動為所有出租中的貨櫃產生租金帳單
4. 表格中：
   - 每筆帳單顯示金額、已繳、餘額、付款方式、到期日
   - 未繳的可以選付款方式後點「標記已繳」
   - 如有部分付款紀錄，會顯示「X 筆 / $總額」，滑鼠移上去可看明細
5. 收據/發票狀態以綠（已開立）、黃（待開立）、藍（免開立）標籤顯示

---

## 七、查看電費

1. 左側選單點「電費管理」
2. 表格顯示每個貨櫃的上期/本期電表度數、用電量、金額
3. 基本電費每年 1,000 元，大量用電每度 9 元

---

## 八、退租試算

1. 左側選單點「退租管理」
2. 選擇要退租的貨櫃
3. 設定退租日期
4. 可輸入損壞扣款、清潔費、違約金（沒有的話保持 0）
5. 系統自動計算：
   - 已用/剩餘月份
   - 剩餘租金退費
   - 扣款合計
   - 最終應退或最終應補的金額
6. 若確定退租，點「完成退租」按鈕 → 貨櫃會自動變回空櫃，合約關閉，客戶狀態更新

---

## 九、使用 AI 助理

1. 左側選單點「AI 助理」，或點右下角浮動 AI 按鈕
2. 輸入問題或點選建議問題，例如：
   - 「目前有多少空櫃？」
   - 「誰還沒繳錢？」
   - 「逾期金額多少？」
   - 「今天要處理什麼？」
3. 系統會根據現有資料回答

---

## 十、匯出與匯入資料

### Export JSON
1. 回到 Dashboard，找到「資料管理」區塊
2. 點「Export JSON」→ 下載完整資料備份檔

### Import JSON
1. 點「Import JSON」按鈕 → 選擇之前下載的 `.json` 檔案
2. 系統會檢查格式，成功後會更新所有資料
3. 若匯入檔案缺少部分欄位，系統會提醒並用預設資料補齊

### Export CSV
1. 點「CSV: Customers」、「CSV: Contracts」或「CSV: Payments」
2. 下載的 CSV 可用 Excel 直接開啟（已含中文支援）

### Reset Demo Data
1. 點「Reset Demo Data」
2. 系統會詢問是否確定 → 確認後清除所有資料，恢復為初始展示資料

---

> 提示：所有修改都會自動儲存到瀏覽器。關閉網頁後重新開啟，資料不會消失。
